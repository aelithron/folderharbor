# FolderHarbor Web
This is a FolderHarbor client for file management through the web!
## Usage
Proper usage guide coming soon!

This client will automatically configure itself to the FolderHarbor server's settings.
### Environment Variables
- `SERVER_URL`: The URL to a FolderHarbor server, formatted as `http[s]://[subdomain.]<domain>.<tld>` (with no slash at the end).
