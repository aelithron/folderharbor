# FolderHarbor Web
This is a web client for FolderHarbor administration! It cannot be run on its own, and requires an operational FolderHarbor server.

### Environment Variables
You should set the `NEXT_PUBLIC_DEFAULT_URL` environment variable to your FolderHarbor server's URL. This is in the format of `http(s)://[domain]:[port]`. For example, `https://demo.folderharbor.novatea.dev:3000`. In development, this wil likely be `http://localhost:3000`. Note that the user can override this inside of the UI, this is the default URL.
